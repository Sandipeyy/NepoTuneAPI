## ğŸš€ Quick Start

Clone the repository:

```sh
git clone https://github.com/Sandipeyy/NepoTuneAPI
cd NepoTuneAPI
```

### Using Docker

```sh
docker-compose up
```

### Or Run Manually

> **Note:** Requires `Bun (1.0.29+)` or `Node.js (v20+)`

```sh
bun install
bun run dev
```

Or with Node.js:

```sh
npm install
npm run dev
```
